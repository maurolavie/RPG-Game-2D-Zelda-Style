# RPG-Game-2D-Zelda-Style
RPG game with free tiles from a tutorial for practice how to create a video game. Includes animation, characters, player and enemies movements, C# scripts,collisions, physics, teletransport between different zone, atack with sword, lightning with sword, destroyable objects, enemys with raycast like a radar to detect player and follow to attack, enemy atacks, life of enemys.

English description - How to play

use the arrows key to move the player
press space key to atack with sword you can defeat enemies
press alt key and mantain to charge the lightning and unpress to fire then you can broken objects
explore three diferents zone starting in test zone is like the player comes from some travel, found the next one and enemies apears to attack,
at the end you can go to the house.


Descripción en español - Cómo jugar

usa la tecla de flechas para mover al jugador
presione la tecla de espacio para atacar con la espada, puede derrotar a los enemigos
presione la tecla alt y manténgase para cargar el rayo y presione para disparar, luego puede romper objetos
explorar tres zonas diferentes comenzando en la zona de prueba es como si el jugador viniera de algún viaje, encontrara el siguiente y aparecieran enemigos para atacar,
al final puedes ir a la casa.
